Network Fundamentals

In this section, we learned how the security team should focus on the fundamental concepts of networks and have a high-level understanding of the TCP/IP and OSI 7-layer model. This section provides an excellent understanding of the communication layer between the infrastructure environment and the different network layers. TCP/IP is the central part of establishing any communication and encapsulate the segment into the IP packet. There are several TCP and UDP ports assigned to known protocols. It is essential from the security level to allow the required port and ensure all the data flows through the network are encrypted to avoid any data spoofing.

As part of the network security, there is a number of basic standards to protect our network from any attack. As an example of that, the firewall team should deny any ICMP from internet access and internally between the devices connected to the web to avoid any ICMP attack. Also, SSH port should just be enabled from limited IP address subnets like the management network. Therefore, understanding the OSI model can give IT engineers an extensive sight of infrastructure security.
